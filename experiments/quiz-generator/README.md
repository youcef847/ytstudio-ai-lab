# Quiz Generator Experiment

This experiment tests whether AI can generate high-quality Algerian history quiz questions in a structured, reviewable format.

The goal is not to publish directly, but to produce draft content that a human can review and approve.

## What this experiment does

- AI is given a clear prompt
- AI generates quiz questions in JSON format
- The output is placed in `/output`
- A human reviews and decides what to keep

## Success criteria

This experiment is successful if:
- The JSON is valid
- The questions are historically correct
- The options are fair and not obvious
- The content fits an Arabic quiz app
